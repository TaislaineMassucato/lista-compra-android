# 🛍️ Lista de Compras Android

Aplicativo simples de lista de compras desenvolvido em **Kotlin** no Android Studio.

Permite adicionar, marcar, editar e remover itens da lista de forma rápida e visual.

---

## ✨ Funcionalidades

* ➕ Adicionar itens
* ✅ Marcar como comprado
* ✏️ Editar item
* 🗑️ Remover item
* 🕒 Data e hora automática
* 🎨 Interface moderna com MaterialCard

## 🚀 Como rodar o projeto

### ✅ Pré-requisitos

* Android Studio instalado
* JDK 17+
* Gradle compatível
* Emulador ou celular Android

---

### 🔧 Passo a passo

1. Clone o repositório:

```bash
git clone https://github.com/SEU_USUARIO/lista-compra-android.git
```

2. Abra no Android Studio:

```
File → Open → selecione a pasta do projeto
```

3. Aguarde o **Gradle Sync**

4. Execute o app:

```
Run ▶ app
```
